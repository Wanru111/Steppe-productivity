# Repository Contents

This repository contains code and data used to analyze extreme tree-ring growth and its relationship with reconstructed NDVI.

## Files

- **EGR.txt**  
  Code for calculating the **Extreme Growth Ratio (EGR)** from tree-ring width series.

- **Granger.R**  
  R script for **time-varying Granger causality analysis** between EGR, NDVI, and/or related variables.

- **ndvi_reconstruction.xlsx**
